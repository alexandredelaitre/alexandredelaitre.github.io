Hydra Wallet

Find the Github repository here.

Hydrawallet was something I developed between a partnership with MLH (Major League Hacking) and SolanaLabs.

It was a dApp that allowed teams to engage in trustless profit sharing so that if a project they made earned any tokens, it would be automatically streamed to the individuals wallets based on agreed percentages.
