QuestMaster

Find the Github repository here.

QuestMaster was submitted to DurHack2022 (at Durham University) and won the Best Use of DeSo award!

The idea of QuestMaster was that someone may know what inputs and outputs they want for a program but they might not know how to get from one to the other... We also recognised that people might choose a more expensive freelance developer as they are unsure of the quality of a cheaper freelance developer...
So we levelled the playing field! Through the use of DeSo, we make this request public and allow any developer to complete the task (there is a bounty for completion so unfair jobs will not be completed). We have a verification function to ensure that the code does exactly as expected with new input data so it is not faked and now everyone is happy! The quest giver has gotten the code they were looking for and the bounty hunter has been paid!

This creates a bounty system for freelancing that anyone can partake in.
